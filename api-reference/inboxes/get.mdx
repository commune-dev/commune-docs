---
title: "Get inbox"
api: "GET /v1/domains/:domainId/inboxes/:inboxId"
description: "Retrieve a single inbox by domain and inbox ID."
---

<RequestExample>
```typescript TypeScript
import { CommuneClient } from 'commune-ai';

const commune = new CommuneClient({ apiKey: process.env.COMMUNE_API_KEY });

const inbox = await commune.inboxes.get('3d4f5a6b-7c8d-9e0f-a1b2-c3d4e5f6a7b8');

console.log(inbox.data.address);      // support@mycompany.com
console.log(inbox.data.displayName);  // Support Agent
```
```python Python
from commune import CommuneClient

client = CommuneClient()

inbox = client.inboxes.get("3d4f5a6b-7c8d-9e0f-a1b2-c3d4e5f6a7b8")

print(inbox.data.address)
```

## Path Parameters

<ParamField path="domainId" type="string" required>
  The domain ID the inbox belongs to.
</ParamField>

<ParamField path="inboxId" type="string" required>
  The inbox ID (UUID) to retrieve.
</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="properties" defaultOpen>
    <ResponseField name="id" type="string">
      Unique inbox identifier (UUID).
    </ResponseField>
    <ResponseField name="localPart" type="string">
