## Path Parameters

<ParamField path="domainId" type="string" required>
  The domain ID the inbox belongs to.
</ParamField>

<ParamField path="inboxId" type="string" required>
  The inbox ID (UUID) to update.
</ParamField>

## Body

All body fields are optional. Only provided fields are updated; others retain their existing values.

<ParamField body="local_part" type="string">
  New local part for the inbox address. Accepts either `local_part` or `localPart`.
</ParamField>

<ParamField body="webhook" type="object">
  Updated webhook configuration. Replaces the entire existing webhook config.

  <Expandable title="properties">
    <ParamField body="endpoint" type="string">
      HTTPS URL to receive webhook events.
    </ParamField>
    <ParamField body="events" type="string[]">
      Event types to subscribe to. Example: `["inbound"]`.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="status" type="string">
  New inbox status. Example values: `"active"`, `"inactive"`.
</ParamField>

## Response

<ResponseField name="data" type="object">
  The updated inbox object. Same shape as [Get inbox](/api-reference/inboxes/get).

  <Expandable title="properties" defaultOpen>
    <ResponseField name="id" type="string">
      Inbox ID (unchanged).
    </ResponseField>
    <ResponseField name="localPart" type="string">
      Updated local part of the inbox address.
    </ResponseField>
    <ResponseField name="address" type="string">
      Full email address of the inbox.
    </ResponseField>
    <ResponseField name="displayName" type="string">
      Display name shown in outbound `From` headers.
    </ResponseField>
    <ResponseField name="webhook" type="object">
      <Expandable title="properties">
        <ResponseField name="endpoint" type="string">Webhook endpoint URL.</ResponseField>
        <ResponseField name="events" type="string[]">Subscribed event types.</ResponseField>
        <ResponseField name="secret" type="string">HMAC signing secret for payload verification.</ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="status" type="string">
      Current inbox status.
    </ResponseField>
    <ResponseField name="createdAt" type="string">
      ISO 8601 creation timestamp.
    </ResponseField>
  </Expandable>
</ResponseField>
