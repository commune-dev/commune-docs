The Commune API lets your agent send and receive email and SMS, manage inboxes and phone numbers, and handle multi-turn conversations with real users — all over standard HTTP.

## Base URL

```
https://api.commune.email/v1
```

All endpoints are served over HTTPS. HTTP requests are rejected.

## Authentication

All requests require an API key passed as a Bearer token:

```bash
Authorization: Bearer comm_your_api_key_here
```

[Get your API key →](/api-reference/authentication)

## Response format

All successful responses return a `data` field:

```json
{
  "data": { ... }
}
```

All errors return an `error` code and `message`:

```json
{
  "error": "validation_error",
  "message": "Human-readable description"
}
```

[Full error reference →](/api-reference/errors)
