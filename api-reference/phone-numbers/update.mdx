```python Python
        "body": "Thanks for your message! We'll respond within 1 business day.",
    },
)

print(updated.friendly_name)  # Customer Support
```

## Path Parameters

<ParamField path="id" type="string" required>
  The phone number ID. Format: `pn_...`
</ParamField>

## Body

All fields are optional. Only provided fields are updated.

<ParamField body="friendly_name" type="string">
  Display name for this number (max 100 characters).
</ParamField>

<ParamField body="webhook" type="object">
  Webhook configuration for inbound and outbound SMS events.

  <Expandable title="properties">
    <ParamField body="endpoint" type="string">
      HTTPS URL where Commune will POST SMS events. Must be a valid URL.
    </ParamField>
    <ParamField body="secret" type="string">
      Signing secret used to generate the `X-Commune-Signature` header. Store this securely and verify it on your server to confirm payloads originate from Commune.
    </ParamField>
    <ParamField body="events" type="string[]">
      Array of event types to send to this webhook. Examples: `sms.received`, `sms.sent`, `sms.status_updated`.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="auto_reply" type="object">
  Automatic reply sent to every inbound message.

  <Expandable title="properties">
    <ParamField body="enabled" type="boolean" required>
      Whether auto-reply is active.
    </ParamField>
    <ParamField body="body" type="string" required>
      The auto-reply message text (max 1600 characters). Credits are charged per auto-reply sent.
    </ParamField>
  </Expandable>
</ParamField>

## Response

<ResponseField name="data" type="object">
  The updated phone number record. See [Get Phone Number](/api-reference/phone-numbers/get) for the full response schema.
</ResponseField>

<Note>
**Webhook signing:** When a webhook `endpoint` is set, Commune sends an `X-Commune-Signature` header with each request. The signature is an HMAC-SHA256 of the request body using your webhook `secret`. Verify this header on your server to confirm the request is authentic.

```
X-Commune-Signature: sha256=abc123...
```
</Note>

<Note>
**Auto-reply credits:** Each auto-reply consumes the same credits as a regular outbound SMS (2 credits per segment for US numbers).
</Note>
